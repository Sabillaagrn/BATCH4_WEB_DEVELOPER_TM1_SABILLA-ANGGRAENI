Data Analyst

Deskripsi Program: Program ini dirancang untuk membantu peserta memahami dan menguasai keterampilan analisis data yang diperlukan untuk membuat keputusan berbasis data. Peserta akan mempelajari teknik-teknik analisis data, penggunaan alat-alat statistik, dan pengolahan data menggunakan perangkat lunak analisis data populer seperti Excel, SQL, dan Python.

Biaya: Rp. 6.000.000

Kurikulum:

  • Minggu 1-2: Pengenalan Analisis Data - Konsep dasar dan teknik analisis data
  • Minggu 3-4: Penggunaan Excel untuk Analisis Data - Fungsi, rumus, dan teknik analisis data di Excel
  • Minggu 5-6: SQL untuk Analisis Data - Query dasar dan lanjutan untuk pengolahan data
  • Minggu 7-8: Visualisasi Data - Teknik visualisasi data menggunakan alat seperti Tableau dan Power BI
  • Minggu 9-10: Pengantar Python untuk Data Analysis - Penggunaan Python dalam analisis data dan pemrograman dasar
  • Minggu 11-12: Statistik untuk Data Analysis - Konsep statistik dasar dan penerapannya dalam analisis data
  • Minggu 13-16: Proyek Akhir - Penerapan keterampilan analisis data dalam proyek nyata dan presentasi hasil

Jadwal Kelas:

Kelas dilaksanakan setiap Senin dan Rabu, pukul 18.00 - 21.00 WIB.

Tempat: SmartTech, Jl. Teknologi No. 45, Jakarta

Informasi Tenaga Pelatih:

Alex Johnson

Pengalaman: 10 tahun dalam analisis data, spesialisasi dalam Excel, SQL, dan Python.

Emily Davis

Pengalaman: 8 tahun dalam visualisasi data dan penggunaan alat-alat analisis data seperti Tableau dan Power BI.

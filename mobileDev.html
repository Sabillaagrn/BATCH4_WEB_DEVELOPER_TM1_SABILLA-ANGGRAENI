Mobile Development

Deskripsi Program: Program Mobile Development kami dirancang untuk memberikan pemahaman komprehensif tentang pengembangan aplikasi mobile untuk platform iOS dan Android. Peserta akan belajar tentang desain responsif, pengembangan aplikasi menggunakan Swift dan Kotlin, serta integrasi dengan backend dan API.

Durasi: 4 Bulan

Biaya: Rp 6.000.000

Kurikulum:

  • Minggu 1-2: Pengenalan Swift dan Kotlin - Dasar-dasar pemrograman untuk iOS dan Android
  • Minggu 3-4: Desain UI/UX Mobile - Prinsip desain untuk aplikasi mobile
  • Minggu 5-6: Pengembangan Aplikasi iOS - Penggunaan Xcode dan Swift
  • Minggu 7-8: Pengembangan Aplikasi Android - Penggunaan Android Studio dan Kotlin
  • Minggu 9-10: Backend Integration - Menghubungkan aplikasi dengan backend dan API
  • Minggu 11-12: Testing dan Debugging - Teknik untuk menguji dan memperbaiki aplikasi
  • Minggu 13-16: Proyek Akhir - Penerapan semua materi dalam proyek aplikasi mobile

Jadwal Kelas:

Kelas dilaksanakan setiap Selasa dan Kamis, pukul 18.00 - 21.00 WIB.

Tempat: SmartTech, Jl. Teknologi No. 45, Jakarta

Informasi Tenaga Pelatih:

Alex Johnson

Pengalaman: 12 tahun dalam pengembangan aplikasi mobile, spesialisasi dalam Swift dan Kotlin.

Emily Davis

Pengalaman: 9 tahun dalam desain UI/UX mobile dan pengembangan aplikasi Android dan iOS.
